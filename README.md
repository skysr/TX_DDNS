# TX_DDNS

这是一个用于**动态DNS更新**的Shell脚本，主要功能是将本地网络的IPv6地址自动同步到DNS服务商（dnsapi.cn），确保域名始终指向正确的IP地址。

**核心特点：**
- 从网卡（enpxsx）获取当前公网IPv6地址
- 对比新旧IP，仅在地址变化或达到60分钟强制更新周期时触发更新
- 使用dnspod.cn API接口修改DNS记录
- 通过本地日志文件（`/var/log/ddns.db` 和 `/var/log/ddns.flage`）记录IP状态和更新频次

**工作流程：**
1. 提取网卡IPv6地址 → 2. 读取历史记录 → 3. IP比对与计时判断 → 4. 条件满足时调用API更新 → 5. 记录新IP到本地

**适用场景：** 家庭宽带IPv6动态IP环境下的远程访问服务。
